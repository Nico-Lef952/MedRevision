## 📌 À retenir absolument (incontournables +++)

> 💡 **Asthénie + anorexie + amaigrissement = CANCER** jusqu'à preuve du contraire +++
💡 **Méléna** = sang digéré noir = hémorragie **haute** | **Rectorragie** = rouge vif = hémorragie **basse** +++
💡 **Défense** = contraction INVOLONTAIRE vincible → irritation péritonéale | **Contracture** = invincible + permanente → **péritonite** +++
💡 **Signe de Blumberg** = douleur à la **décompression brutale** (appuyer puis lâcher vite) → irritation péritonéale +++
💡 **Hernie** = orifice **naturel** | **Éventration** = ancienne **incision chirurgicale** | **Éviscération** = viscères qui sortent littéralement +++
💡 **Manœuvre de Murphy** = palpation HCD + inspiration profonde → douleur qui **bloque l'inspiration** = vésicule biliaire atteinte +++
💡 **Appendicite = urgence abdominale la plus fréquente** → surtout avant 30 ans, prédominance masculine +++
💡 **Point de McBurney** = union **1/3 externe / 2/3 internes** de la ligne EIAS-ombilic +++
💡 **Signe de Rovsing** = pathognomonique appendicite : douleur en **FID** lors de palpation de la **FIG** +++
💡 **β-HCG systématique** chez toute femme en âge de procréer avec douleur abdominale → éliminer **GEU** +++
💡 **Jamais de pneumopéritoine** dans la péritonite appendiculaire **QCM+++** +++
💡 **Péritonite sur ulcère** = **PAS de fièvre** au début (péritonite chimique) + **PAS d'hyperleucocytose** au début +++
💡 **Fibroscopie CONTRE-INDIQUÉE** dans la perforation d'ulcère (injecte de l'air → aggrave) +++
💡 **Carré de Mondor** de l'occlusion = douleur + vomissements + arrêt transit + météorisme +++
💡 **Signe le plus spécifique** de l'occlusion = **arrêt des GAZ** (les selles peuvent persister temporairement) +++
💡 **Occlusion haute** = vomissements précoces + AMG tardif + météorisme minime | **Occlusion basse** = inverse +++
💡 **Strangulation** = début **brutal** + silence auscultatoire | **Obstruction** = début **progressif** + bruits hydro-aériques +++
💡 **Occlusion grêle sur bride** = cause la **plus fréquente** des occlusions + tout ATCD chirurgical +++
💡 **Volvulus sigmoïde** = sujet âgé + constipé + météorisme asymétrique oblique + vacuité FIG +++
💡 **Cancer côlon** = cause la plus fréquente d'occlusion colique par **obstruction** +++
💡 **Perforation diastaltique** = si côlon dilaté > **10 cm** au-dessus obstacle → risque de perforation du caecum +++
💡 **Anse iléale dilatée** > **25 mm** | **Côlon dilaté** > **7–8 cm** au scanner +++
💡 **ECG systématique** chez tout patient > 50 ans avec douleur abdominale (éliminer IDM) +++

## 1️⃣ L'Examen Abdominal — Méthodologie

### A) Repères anatomiques — 9 cadrans de l'abdomen

| Cadran | Contenu |
| --- | --- |
| **Hypochondre droit** | Foie, vésicule, voies biliaires |
| **Épigastre** | Estomac, duodénum, pancréas |
| **Hypochondre gauche** | Rate, queue du pancréas, angle colique gauche |
| **Flanc droit** | Côlon droit, voies urinaires droites |
| **Région ombilicale** | Intestin grêle, côlon transverse, mésentère |
| **Flanc gauche** | Côlon gauche, voies urinaires gauches |
| **Fosse iliaque droite (FID)** | Appendice, trompe droite |
| **Hypogastre** | Vessie, endomètre |
| **Fosse iliaque gauche (FIG)** | Sigmoïde, trompe gauche |

### B) Interrogatoire — Méthode policière

- **ATCD à rechercher** :
    - Personnels : médicaux + chirurgicaux (cicatrices = bride/éventration possible !)
    - Familiaux
    - Allergies, intoxications (alcool, tabac, toxicomanie)
    - Traitements : **gastrotoxiques** (aspirine, AINS → ulcères) + **hépatotoxiques** (paracétamol)
- **Caractérisation de la douleur (STOPIARF)** :
    
    
    | Élément | Ce qu'on cherche |
    | --- | --- |
    | **Siège** | Quadrant + irradiations |
    | **Type** | Crampe, brûlure, pesanteur, sourd, aigu |
    | **Onset** (mode d'apparition) | Brutal (perforation, strangulation) ou progressif (inflammation) |
    | **Intensité** | EVA 1–10 + évolution croissante ? |
    | **Ancienneté** | Aiguë ou chronique |
    | **Rythme** | Permanent, intermittent, récurrent |
    | **Facteurs** influençants | Position antalgique ? alimentation ? gaz ? |
- **Positions antalgiques** :
    - **Chien de fusil** → douleur **pancréatique**
    - **Patient qui s'agite** → **colique néphrétique**

### C) Signes Accompagnants

| Signe | Signification |
| --- | --- |
| **Asthénie + anorexie + amaigrissement** | **CANCER** +++ |
| **Nausées/vomissements** | Fréquence, aspect (alimentaire → bilieux → fécaloïde) |
| **Transit** | Diarrhée, constipation, arrêt des matières et GAZ |
| **Météorisme** | "Ballon d'air" = occlusion |
| **Hématémèse** | Vomir du sang |
| **Méléna** | Sang digéré = noir = hémorragie **haute** (sang dans les selles) |
| **Rectorragie** | Rouge vif, recouvre les selles = hémorragie **basse** (hémorroïdes, côlon) |
| **Ictère, pâleur** | — |
| **Signes gynéco** | Métrorragie, leucorrhées |
| **Signes urinaires** | Brûlures mictionnelles |
| **Dysphagie** | Sténose œsophagienne |

## 2️⃣ Examen Physique — Les 4 Temps

### A) Règles générales

- Patient en **décubitus dorsal**, genoux fléchis si possible (relâche la paroi)
- Déshabillé de **l'appendice xiphoïde au pubis** + régions inguinales
- Médecin à **droite** du patient, mains réchauffées
- Maintenir le **contact visuel** (surveiller mimiques douloureuses)

### B) Inspection

| Élément | Ce qu'on cherche | Signification |
| --- | --- | --- |
| **Faciès** | Pâleur (anémie), ictère, gris (péritonite) | — |
| **Position antalgique** | Chien de fusil, agitation | Pancréas, colique néphrétique |
| **Cicatrices** | Abdomen, ombilic (coelioscopies+++) | Bride, éventration possible |
| **Distension abdominale** | Asymétrique ou diffuse | Ascite, occlusion |
| **Voussure, tuméfaction** | Effort de poussée (Valsalva = tousser ou boucher le nez et pousser) | Hernie ombilicale, inguinale |
| **Veines dilatées** | Réseau veineux visible | Circulation collatérale = HTP |
| **Péristaltisme visible** | Ondulations des anses | Patient maigre + occlusion |

### C) Auscultation

| Signe | Valeur | Signification |
| --- | --- | --- |
| **Bruits hydro-aériques normaux** | 5–35/min, toutes les 5–15 secondes | Péristaltisme normal |
| **Hyperactivité** | Très nombreux | Intestin lutte contre un obstacle (obstruction) |
| **Silence auscultatoire** | Absence totale | Intestin paralysé (iléus, strangulation, péritonite) |
| **Souffles vasculaires** | À ausculter | FdR cardiovasculaires |

### D) Percussion

| Son | Signification |
| --- | --- |
| **Tympanisme** | Organe creux (air) |
| **Matité** | Organe plein (foie, rate, vessie pleine) ou ascite |
| **Disparition matité hépatique** | Pneumopéritoine volumineux |
| **Matité déclive des flancs** | Épanchement liquidien intrapéritonéal abondant (ascite) |

> **Technique** : main dominante, choc brusque du poignet avec le majeur frappant la 2e phalange du majeur posé à plat.
> 

### E) Palpation

**Règle** : toujours commencer **à distance du site douloureux**, cadran par cadran, une à deux mains à plat.

- **Superficielle** → zone sensible
- **Profonde** → recherche de masse (hépatomégalie, splénomégalie, tumeur)

**Signes d'irritation péritonéale** :

| Signe | Mécanisme | Gravité |
| --- | --- | --- |
| **Défense** | Contraction douloureuse **INVOLONTAIRE** vincible à la palpation profonde | Inflammation péritonéale localisée |
| **Contracture** | Contraction involontaire **invincible et permanente** = "ventre de bois" | **Péritonite** (signe de gravité) |
| **Signe de Blumberg** | Exacerbation de la douleur à la **dépression soudaine** (appuyer + lâcher brutalement) | Irritation péritonéale |

> 💡 La défense est **toujours localisée là où le problème commence** → valeur localisatrice +++.
>

## 3️⃣ Intégrité Pariétale +++

- Définitions à connaître (QCM +++)
    
    
    | Terme | Définition | Différence clé |
    | --- | --- | --- |
    | **Hernie** | Passage anormal du contenu abdominal à travers un **orifice naturel** de la paroi | Zone de faiblesse **congénitale** |
    | **Éventration** | Passage anormal à travers une **ancienne incision chirurgicale** | **Impossible sans ATCD chirurgical** |
    | **Éviscération** | Viscères qui sortent **littéralement** de la paroi (couteau, post-op précoce) | Visible ou sous peau mais sans péritoine |
    
    **Collet herniaire** = les berges de la hernie | **Sac herniaire** = le bout d'intestin sorti.
    
    **Complications** : la hernie peut entraîner une **occlusion** (si collet petit → risque d'étranglement ++).
    
- Orifices herniaires à explorer
    
    
    | Orifice | Localisation | Terrain |
    | --- | --- | --- |
    | **Ombilical** | Ombilic | Enfants +++ + femmes enceintes après accouchement |
    | **Ligne blanche** | Entre xiphoïde et pubis | — |
    | **Inguinal** | Au-dessus de la ligne de Malgaigne | — |
    | **Crural** | En dessous de la ligne de Malgaigne | — |
    
    **Technique** : examen allongé + debout + manœuvre de Valsalva (toux ou pousser en bouchant le nez).
    
    **Hernie inguinale** : invagination du scrotum avec index en partant du bas du scrotum → cordon spermatique → anneau inguinal superficiel.
    
    > ⚠️ Toujours vérifier **les DEUX côtés** (hernies très souvent bilatérales).
    💡 Une hernie n'est pas forcément douloureuse (juste une boule) → on n'opère pas si asymptomatique.
    > 
- Manœuvre de Murphy
    
    **Technique** : palper l'hypochondre droit sous le rebord costal → demander au patient une **inspiration profonde**.
    
    **Positif** : la douleur au site de palpation **empêche le patient de terminer son inspiration** = vésicule biliaire atteinte.
    
    > ⚠️ ≠ **Signe de Murphy radiologique** : douleur lors de l'écho quand on explore la vésicule avec la sonde.
    >

## 4️⃣ Toucher Rectal (TR) et Toucher Vaginal (TV)

- Toucher Rectal
    
    **Conditions** : consentement du patient + expliquer le geste + noter dans le CR si non fait.
    
    **Position** : décubitus **LATÉRAL** genoux sur la poitrine (position fœtale) OU gynécologique. **PAS à 4 pattes.**
    
    **Technique** : rotation 360° du doigt + effort de poussée du patient (fait descendre le rectum).
    
    **On recherche** :
    
    - **Tonus du sphincter anal**
    - **Fécalome** (très fréquent personnes âgées) + tumeur
    - **Présence de sang** sur le doigtier
    - **Cul-de-sac de Douglas** : comblement = abcès / douleur = **cri du Douglas** (épanchement péritonéal infecté ou agressif)
    - **Prostate** chez l'homme
- Toucher Vaginal
    
    On recherche : douleur à la **mobilisation utérine** + masse latéro-utérine + sang ou leucorrhées.

## 5️⃣ Appendicites Aiguës +++

**Définition** : inflammation aiguë de l'appendice = **urgence abdominale la plus fréquente**.

**Terrain** : tout âge mais **surtout avant 30 ans**, prédominance **masculine**. Rare avant 3 ans et chez la personne âgée → formes compliquées plus fréquentes dans ces populations.

### Physiopathologie

Obstruction de la lumière appendiculaire → pullulation microbienne → lésions muqueuses puis pan-pariétales.

**Causes d'obstruction** :

- **Stercolithe** : résidus de matières fécales calcifiées (visible à l'écho, ASP, scanner)
- **Hyperplasie lymphoïde** : infections virales ou bactériennes
- **Bouchon muqueux, corps étranger, parasite** (oxyures, ascaris)
- **Inflammation pariétale** : maladie de Crohn

### Signes Fonctionnels

- **Fébricule** : 37,5–38,5°C (petite fièvre)
- **Douleur progressive en FID**, sans irradiation, augmentant sur 24h
- Début parfois **épigastrique** → migration progressive en FID
- Signes associés (iléus réflexe) : nausées (50%), vomissements (30%), ralentissement transit (30%)

> ⚠️ Si douleur **brutale** → plutôt **perforation** | Si fièvre **> 40°C** → infection urinaire ou péritonite.
> 

### Signes Physiques +++

**Point de McBurney** : union **1/3 externe / 2/3 internes** de la ligne joignant l'EIAS et l'ombilic.

| Signe | Technique | Signification |
| --- | --- | --- |
| **Douleur provoquée / défense FID** | Palpation FID (mieux avec jambe pendante hors du lit) | Irritation péritonéale localisée |
| **Signe de Blumberg** | Décompression brutale FID | Irritation péritonéale |
| **Signe de Rovsing** | Douleur en **FID** lors de palpation de la **FIG** | **Pathognomonique** de l'appendicite +++ |
| **Douleur latérale droite au TR** | Toucher rectal | Irritation péritonéale |

### Paraclinique

**Biologie** :

- **NFS** : hyperleucocytose à PNN (inflammation)
- **CRP** : élevée > 8
- **β-HCG** : SYSTÉMATIQUE chez toute femme en âge de procréer → éliminer **GEU** !!!
- **BU** : éliminer **infection urinaire** !!!
- Coagulation + groupe sanguin (si chirurgie)

**Imagerie** :

- **ASP** : aucune place sauf stercolithes radio-opaques
- **Échographie** : 1ère intention, à jeun + vessie pleine → chez femme enceinte (élimine GEU, torsion annexes, salpingite)
- **Scanner abdomino-pelvien injecté** : à jeun + allergies + créatinine + grossesse ?

### Formes Cliniques — Appendice Ectopique +++

| Forme | Localisation | Présentation clinique |
| --- | --- | --- |
| **Latéro-caecale** | Position typique | Tableau classique FID |
| **Pelvienne** | Hypogastre droit | Douleurs hypogastre droite + signes urinaires (pollakiurie, dysurie) ou rectaux (ténesme) → BU négative + écho normale |
| **Sous-hépatique** | Sous le foie (caecum haut ou appendice long) | Mimic cholécystite : fièvre + douleur + défense HCD → écho élimine cholécystite |
| **Rétro-caecale** | Derrière le caecum, devant le psoas | Douleurs **lombaires droites** + psoïtis (flexion-adduction cuisse) soulage la douleur + **pas de défense FID** (caecum bloque) + douleur FID à l'élévation de la jambe |
| **Méso-cœliaque** | Milieu des anses | **Iléus réflexe fébrile** → tableau d'occlusion fébrile → diagnostic par scanner |
| **Gauche** | Mésentère commun / situs inversus | Scanner indispensable |

### Traitement

**Appendicectomie** (chirurgical) :

- Voie d'abord au **point de McBurney** → enfant de sexe masculin
- Voie par **cœlioscopie** → femmes, obèses

## 6️⃣ Péritonites Aiguës +++

**Définition** : inflammation ou infection aiguë du **péritoine** → peut entraîner le décès → diagnostic et traitement chirurgical précoce = pronostic.

### Étiologies

**Perforation d'organe creux** :

- Ulcères gastro-duodénaux, cancers
- Occlusion du grêle sur bride, volvulus, diverticulite sigmoïdienne, Crohn
- Diverticule de Meckel, infarctus mésentérique, colite aiguë grave
- Traumatismes, iatrogenèse (endoscopie)

**Diffusion d'une collection suppurée** :

- Appendicite, diverticulite sigmoïdienne (les plus fréquentes)

### Clinique Générale +++

**Signes généraux** :

- **Fièvre élevée 39–40°** (absente dans les péritonites chimiques = ulcère perforé)
- Altération du faciès (gris)
- Polypnée + tachycardie → chercher signes de **choc**

**Signes fonctionnels** :

- Douleur abdominale **intense, spontanée, continue, généralisée**
- Siège initial = valeur localisatrice
- Mode de début : **brutal** ("coup de poignard") si perforation d'organe creux | **progressif** si diffusion depuis foyer
- Vomissements + **arrêt des matières et des gaz** (iléus réflexe à l'irritation péritonéale)

**Signes physiques** :

- **Inspection** : malade pâle/gris + figé + **abdomen immobile** + Ø respiration abdominale
- **Palpation** : douleur diffuse + hyperesthésie cutanée + **contracture diffuse** ("ventre de bois") = invincible, permanente, diffuse (même sous anesthésie)
- **Percussion** : disparition matité hépatique (pneumopéritoine) OU matité déclive des flancs (épanchement)
- **Auscultation** : **silence auscultatoire** (iléus réflexe)
- **TR** : **cri du Douglas** = douleur à la palpation du cul-de-sac de Douglas = épanchement péritonéal infecté (pus, bile)

> ⚠️ **Personnes âgées, dénutris, immunodéprimés, obèses** : signes péritonéaux peuvent être **modérés voire absents** = péritonite asthénique → vigilance +++.
> 

**Paraclinique** :

- Bio : hyperleucocytose PNN + ↑ CRP + ↑ **PCT** (procalcitonine = marqueur d'infection bactérienne)
- NFS + hémostase + groupe sanguin + ionogramme + **hémocultures** ++
- **TDM avec injection** = examen de référence : pneumopéritoine + épanchement + cause

### Comparatif des 3 Grandes Péritonites +++

|  | **Péritonite appendiculaire** | **Péritonite ulcère gastroduodénal** | **Péritonite diverticulaire** |
| --- | --- | --- | --- |
| **Début** | Progressif | **Brutal** | Progressif+++ ou brutal |
| **Max des signes** | Fosse iliaque **droite** | **Épigastre** | Fosse iliaque **gauche** |
| **Fièvre / hyperleucocytose** | Oui | **Non au début** | Oui |
| **Pneumopéritoine** | **NON** (+++) | Oui, parfois discret | Oui (si perforation) |

### Péritonite sur perforation d'ulcère gastroduodénal

- **Terrain** : ATCD d'ulcère + prise récente d'AINS +++
- **PAS de fièvre** dans les premières heures (péritonite chimique → sucs gastriques = propre)
- **PAS d'hyperleucocytose** au début (infection tardive)
- Douleur épigastrique **brutale** → diffuse à tout l'abdomen
- **Fibroscopie CONTRE-INDIQUÉE** (injecte de l'air dans un organe déjà perforé → aggrave)
- **Traitement** : chirurgie urgence → suture (ulcère duodénal) / excision-suture (ulcère gastrique, pour examen histo) + IPP en TTT de fond

### Péritonite appendiculaire

- **généralisée d’emblée**, ou compliquer un abcès ou un **plastron appendiculaire qui se rompt** secondairement dans le péritoine
- **Jamais de pneumopéritoine** au TDM **QCM+++**
- Sujet jeune + douleur FID progressivement croissante + fébricule → secondairement généralisée + fièvre 39–40°

### Péritonite diverticulaire

- Diverticules = petites hernies de la muqueuse à travers la musculeuse
- **2 types** : par **perforation** (péritonite généralisée brutale = stercorale → choc septique) / par **diffusion** d'un abcès (en 2 temps, progressif)
- Douleur maximale en **FIG** (≈ appendicite à gauche)
- Scanner : pneumopéritoine volumineux (perforation) OU liquide intra-abdominal (purulente)

## 7️⃣ Occlusions Intestinales Aiguës +++

### Physiopathologie

Arrêt du transit (gaz + matières) → stagnation liquidienne + ↓ réabsorption → **3ème secteur liquidien** → **hypovolémie**.

Distension pariétale → douleurs → **ischémie** → **perforation**.
Pullulation microbienne → altération de la paroi intestinale.

### Classification

| Type | Mécanisme | Intestin | Début | Bruits hydro-aériques |
| --- | --- | --- | --- | --- |
| **Fonctionnelle / iléus paralytique** | Intestin perméable mais **paralysé** (pas d'obstacle) | — | Variable | **Silence** |
| **Organique / mécanique** | Obstacle physique | Grêle ou côlon | Variable | Variable |

**Causes d'iléus paralytique** : agression intra/rétropéritonéale (péritonite, appendicite) + post-opératoire + causes métaboliques (hypothyroïdie, hypokaliémie, hypocalcémie, syndrome d'Ogilvie).

**2 mécanismes d'occlusion organique** :

- **Strangulation** → hernie étranglée, volvulus, bride → début **brutal**
- **Obstruction** → tumeur, compression extrinsèque → début **progressif** + beaucoup de bruits hydro-aériques

### Clinique — Le Carré de Mondor

**4 signes caractéristiques** :

1. **Douleurs abdominales**
2. **Vomissements** (alimentaires → bilieux → fécaloïdes)
3. **Arrêt du transit** → le plus important = **arrêt des GAZ** +++
4. **Météorisme** = ballonnement abdominal

> ⚠️ Le signe le plus **spécifique** = arrêt des **gaz** (les selles peuvent temporairement persister si occlusion haute → le segment d'aval se vide).
> 

**Examen physique** :

- Inspection : **météorisme**
- Percussion : **tympanisme**
- Auscultation : silence OU bruits hydro-aériques majorés
- Rechercher : **cicatrices** abdominales (bride/éventration) + **orifices herniaires** (hernie étranglée)
- **TR OBLIGATOIRE** (tumeur ou fécalome)

### Sémiologie selon le Siège de l'Obstacle +++

|  | **Occlusion haute** (duodénum/grêle) | **Occlusion basse** (côlon/rectum) |
| --- | --- | --- |
| **Signes généraux** | Importants, atteinte précoce état général | État général longtemps conservé |
| **Début** | **Brutal**, évolution rapide | **Progressif**, évolution lente |
| **Douleurs** | **Importantes** | Peu intenses |
| **Vomissements** | **Précoces** | **Tardifs** |
| **AMG (arrêt matières/gaz)** | **Tardif** | **Précoce** |
| **Météorisme** | **Minime** (ou absent) | **Important** |

### Sémiologie selon le Mécanisme +++

|  | **Strangulation** | **Obstruction** |
| --- | --- | --- |
| **Signes généraux** | AEG (altération état général) | EG longtemps conservé |
| **Début** | **BRUTAL** (patient connaît l'heure) | **Progressif** |
| **Douleurs** | Vives, permanentes | Par crises |
| **Vomissements** | **Précoces** | Retardés |
| **Météorisme** | Discret (pas le temps) | Diffus |
| **Ondulations** | Absentes | Présentes (patients maigres) |
| **Auscultation** | **Silence auscultatoire** | **Bruits hydro-aériques** |

**Signes de gravité** (ischémie/perforation) :

- Déshydratation
- **Fièvre** : début des complications
- Douleur intense résistant aux antalgiques niveau 1
- **Défense ou contracture** (déjà dans les complications)
→ Traitement symptomatique : **sonde naso-gastrique** pour aspirer l'air et le surplus.

**Syndromes spécifiques** :

- **Syndrome de Koenig** (obstacle grêle partiel) : douleurs abdominales migratrices déclenchées par le repas → aboutissant au même point → cèdent **brutalement** avec gargouillement + bruit de filtration hydro-aérique + débâcle diarrhéique
- **Obstacle colique** : apparition ou aggravation récente d'une **constipation**

### Imagerie — Scanner Abdomino-Pelvien = Examen de Référence

Différencie occlusion **mécanique** vs **fonctionnelle** + donne siège + donne cause.

**Critères scanner** :

| Critère | Valeur normale | Dilatation pathologique |
| --- | --- | --- |
| **Anse iléale** | — | Diamètre > **25 mm** |
| **Côlon** | — | Diamètre > **7–8 cm** |
| **Zone transitionnelle** | — | Jonction intestin plat / intestin dilaté = site de l'obstacle |

### Situations Cliniques — Occlusions Hautes par Strangulation

**3 causes** : brides + hernies + invagination intestinale aiguë.

- Occlusion sur bride +++
    - **Cause la plus fréquente** des occlusions du grêle
    - Tout ATCD de chirurgie abdominale (même minime) → adhérences ou brides fibreuses
    - Peut survenir des **années** après la chirurgie
    - Mécanisme : "capotage" d'une anse au-dessus d'une bride / volvulus d'une anse autour d'un axe fibreux
    - Complication : **ischémie** et **nécrose**
- Hernies étranglées
    - 2e grande cause de strangulation
    - Toutes hernies : inguinale, crurale, ombilicale + Spiegel, obturatrice, hiatus de Winslow
    - Diagnostic clinique : hernie **douloureuse + irréductible**
- Éventrations étranglées
    - Fragilité sur cicatrice chirurgicale → étranglement possible
    - Palper **toutes les cicatrices** +++ pour vérifier l'absence d'orifice

### Situations Cliniques — Occlusions Hautes par Obstruction

**Causes bénignes** : Crohn, sténose radique, endométriose, tuberculose iléo-caecale.

**Causes malignes** : tumeurs du grêle, valvule de Bauhin (polypes, tumeurs carcinoïdes, lymphomes, métastases, adénocarcinomes).

Révélation : syndrome de Koenig (incomplète) ou occlusion complète.

### Situations Cliniques — Occlusions Basses par Strangulation

### Volvulus du sigmoïde +++

- Le plus courant des volvulus du côlon
- **Torsion** de l'anse sigmoïde sur son axe
- Facteur anatomique : **dolicho-méga-côlon** (gros côlon long)
- **Terrain** : sujet âgé, institutionnalisé, peu mobile, constipé chronique, comorbidités (Parkinson, Alzheimer, diabète, psychotropes)
- **Clinique** : météorisme volumineux + **asymétrique** + oblique + **vacuité de la FIG (vide)**
- **Seul cas où on demande une endoscopie** (peut permettre la détorsion)

### Situations Cliniques — Occlusions Basses par Obstruction

**Causes** : cancer côlon gauche + sténose diverticulaire + fécalome + corps étranger.

### Cancer du côlon en occlusion +++

- **Cause la plus fréquente** d'occlusion colique par obstruction
- **Terrain** : > 50 ans + modification récente du transit + rectorragies
- Le plus souvent : côlon **gauche** ou **sigmoïde**
- Scanner : sténose courte, excentrée + angle de raccordement aigu avec côlon sain

**Valvule de Bauhin continente** (physiologique) : dangereux → côlon ne peut pas se décomprimer → distension → **perforation diastaltique du caecum** si diamètre > **10 cm**.

**Valvule de Bauhin incontinente** (sujets âgés ++) : rassurant → l'air peut remonter dans le grêle → pas de risque de perforation colique.

## 8️⃣ Synthèse — Douleur Abdominale Aiguë d'Origine Digestive

**Démarche en 3 étapes** :

1. **Interrogatoire** : caractériser la douleur
2. **Examen clinique** : recueil des signes
3. **Classification** : quel syndrome ?

**4 syndromes à identifier** :

| Syndrome | Signe clé |
| --- | --- |
| **Syndrome occlusif** | Carré de Mondor : douleur + vomissements + arrêt transit + météorisme |
| **Syndrome péritonéal** | Défense (localisée) → contracture (généralisée = ventre de bois) |
| **Syndrome d'ischémie mésentérique** | Douleur profonde intense + syndrome péritonéal ± hémorragie digestive basse → arythmie cardiaque ?, artérite MI ? |
| **Syndrome hémorragique** | Hématémèse, méléna, rectorragie |

> ⚠️ **Toujours penser aux causes extra-digestives** : urinaire, vasculaire, gynécologique, **cardio-thoracique** (ECG systématique > 50 ans avec douleur abdominale).
