# II) ENDOSCOPIE BRONCHIQUE

## Définition

➡️ Introduction d’un **endoscope par une cavité naturelle** → permet la **visualisation directe des bronches**.

- Deux objectifs :
    - **Diagnostique** → anesthésie locale
    - **Thérapeutique** → anesthésie générale (désobstruction…).
- Exploration :
    - passage **entre les cordes vocales → trachée → bronches**
    - jusqu’aux **bronches sous-segmentaires (~6e ordre)**
    - patient **à jeun**, en **décubitus dorsal**
    - voie **nasale ou buccale**.
