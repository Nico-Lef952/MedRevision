## 📌 À retenir absolument

> 💡 **2 types d'immunité** : Innée (rapide, peu spécifique, pas de mémoire) + Adaptative (lente, spécifique, mémoire)
💡 **Cellules sentinelles** = cellules dendritiques · **Cellules patrouilleurs** = lymphocytes
💡 **Cellules de Langerhans** = CPA épidermiques → capturent l'Ag → migrent au ganglion → présentent aux **LT CD4 via CMH II** +++
💡 **CMH I** → présentation aux **LT CD8** (cytotoxiques) · **CMH II** → présentation aux **LT CD4** (helper/régulateurs) +++QCM
💡 **LT CD4 helper** : Th1 (INFγ, IL-2) · Th2 (IL-4, IL-5, IL-13) · Th17 (IL-17, IL-22)
💡 **LT CD4 régulateurs (Treg)** = CD4+CD25+ → produisent **IL-10 et TGFβ** → freinent l'inflammation +++
💡 **LT CD8** → libèrent **perforine** (perfore la membrane) + **granzyme** (détruit l'ADN) → mort cellulaire
💡 **Lymphocytes NK** → éliminent les cellules du soi avec **défaut de CMH** (infection virale ou cancer)
💡 **Eczéma allergique** = médié par un **haptène** → 3 phases : Sensibilisation → Déclenchement → Régulation
💡 **Sensibilisation** = CD capture l'Ag → migre au ganglion → active LT CD4 et CD8 spécifiques
💡 **Déclenchement** = LT spécifiques migrent dans la peau → apoptose des kératinocytes → **"tempête cytokinique"** → eczéma
💡 **Régulation** = recrutement Treg → production **IL-10 + TGFβ** → freinage de la réponse
💡 **Psoriasis/eczéma/dermatite atopique** = CD4 · **Allergie médicamenteuse/Lyell** = CD8 (CMH I)

## 1️⃣ Introduction — Les 2 Types d'Immunité

|  | **Immunité innée** | **Immunité adaptative** |
| --- | --- | --- |
| **Ancienneté** | La + ancienne phylogénétiquement | Plus récente |
| **Rapidité** | **Rapide** (minutes–heures) | **Lente** (jours–semaines) |
| **Spécificité** | **Peu spécifique** | **Très spécifique** (épitope) |
| **Mémoire** | **Pas de mémoire** | **Mémoire** immunologique +++ |
| **Durabilité** | Réponse non durable | Réponse **durable** |
| **Cellules clés** | PNN, Macrophages, NK, Kératinocytes | Lymphocytes T, B, Cellules dendritiques |

> 💡 **Sentinelles** = cellules dendritiques (détectent et capturent l'Ag)
💡 **Patrouilleurs** = lymphocytes (répondent spécifiquement à l'Ag)
>

## 3️⃣ Les Cellules de Langerhans +++

> **= Cellules Présentatrices d'Antigènes (CPA) de l'épiderme** → déclenchent la réponse adaptative à médiation cellulaire
> 
- Étapes de l'action des cellules de Langerhans
    
    ```
    Exoantigène pénètre dans l'épiderme
              │
              ▼
    Capture par la Cellule de Langerhans
    (phagocytose + voie des endosomes)
              │
              ▼
    Expression en surface via molécules du CMH II
              │
              ▼
    Migration dans le système lymphatique
              │
              ▼
    Arrivée au ganglion lymphatique
              │
              ▼
    Présentation de l'Ag aux LT CD4 via CMH II
    + Production de cytokines pro-inflammatoires
              │
              ▼
    ACTIVATION LT CD4+
              │
              ▼
    Réponse immunitaire adaptative à médiation cellulaire
    ```
    
    | Étape | Détail |
    | --- | --- |
    | **Capture de l'Ag** | Par phagocytose et voie des endosomes |
    | **Présentation** | Via molécules du **CMH II** en surface |
    | **Migration** | Système lymphatique → ganglions |
    | **Activation** | Des **LT CD4+** +++ |
    | **Production** | Cytokines **pro-inflammatoires** |
    
    > 💡 Les cellules de Langerhans = **CPA spécialisées de l'épiderme** → elles font le lien entre l'immunité innée (détection du danger) et l'immunité adaptative (activation LT spécifiques) +++QCM
    >

## 4️⃣ Les Lymphocytes

- Généralités
    
    
    | Lymphocyte | Récepteur | Cible |
    | --- | --- | --- |
    | **LT** | **TCR** (T Cell Receptor) | Épitope présenté par CMH |
    | **LB** | **BCR** (B Cell Receptor) | Épitope libre ou présenté |
    
    > 💡 Chaque lymphocyte possède un récepteur **unique et spécifique d'un épitope** → très grande diversité de récepteurs +++
    > 
- LT CD4 Helper = Th (CD4+)
    
    **Activé par une CPA** → se **polarise** selon le contexte cytokinique → 3 sous-types :
    
    | Sous-type | Signal de polarisation | Cytokines produites | Rôle / Pathologies associées |
    | --- | --- | --- | --- |
    | **Th1** | IL-12, IFN-γ, TNF-β | **INFγ, IL-2** | Immunité cellulaire, lutte anti-infectieuse · Psoriasis |
    | **Th2** | IL-4 | **IL-4, IL-5, IL-13** | Immunité humorale, allergie, parasites · Eczéma, dermatite atopique |
    | **Th17** | IL-23 | **IL-17, IL-22** | Inflammation cutanée, défense anti-fongique · Psoriasis |
    
    ```
    MÉMO CYTOKINES +++
    Th1  → INFγ + IL-2          (réponse cellulaire, psoriasis)
    Th2  → IL-4 + IL-5 + IL-13  (allergie, eczéma, DA)
    Th17 → IL-17 + IL-22        (inflammation, psoriasis)
    ```
    
    > 💡 **CD4** impliqués dans : **psoriasis, eczéma, dermatite atopique** +++QCM
    > 
- LT CD4 Régulateurs = Treg (CD4+CD25+)
    
    
    | Caractéristique | Détail |
    | --- | --- |
    | **Phénotype** | CD4+ **CD25+** |
    | **Incapacité** | Ne produit **PAS** d'IL-2 (contrairement aux Th) |
    | **Cytokines produites** | **IL-10** et **TGFβ** (cytokines anti-inflammatoires) |
    | **Mécanisme d'inhibition** | Via **CTLA-4** → inhibe directement les LTh |
    | **Rôle** | Freinage de la réponse immunitaire → évite l'auto-immunité |
    
    > ⚠️ **Treg = frein de l'immunité** → production IL-10 + TGFβ → indispensables pour éviter les réactions auto-immunes et limiter la durée de l'inflammation +++QCM
    > 
- LT CD8 Cytotoxiques (CD8+)
    
    **Activation en 2 signaux obligatoires :**
    
    | Signal | Mécanisme |
    | --- | --- |
    | **1er signal** (stimulation) | Reconnaissance via **CMH I** (exprimé par CPA pro : CD, macrophages, LB / ou cellules du tissu exprimant l'Ag) |
    | **2e signal** (costimulation) | Interaction **CD28/B7** (sur CPA) |
    
    > ⚠️ Si **B7 insuffisamment exprimé** par la CPA → intervention des **LT CD4** nécessaire
    Le contact **CD40/CD40L** (CPA/TCR) fait augmenter l'expression de B7 → activation complète du LT CD8
    > 
    
    **Mécanismes cytotoxiques :**
    
    | Molécule | Mécanisme d'action |
    | --- | --- |
    | **Perforine** | Perforation de la **membrane** de la cellule cible |
    | **Granzyme** | Destruction de l'**ADN** de la cellule cible → apoptose |
    
    ```
    ACTIVATION LT CD8 :
    
    CPA exprime CMH I + Ag
          │
          ▼
    1er signal : TCR du CD8 reconnaît CMH I/Ag
          │
          ▼
    2e signal : CD28 (CD8) + B7 (CPA) → costimulation
    [Si B7 insuffisant → aide du LT CD4 nécessaire]
          │
          ▼
    LT CD8 ACTIVÉ
          │
          ▼
    Libération granules cytotoxiques :
    PERFORINE → perfore la membrane
    GRANZYME  → détruit l'ADN
          │
          ▼
    MORT de la cellule cible (apoptose)
    ```
    
    > 💡 **CD8** impliqués dans : **allergies médicamenteuses, toxidermies, syndrome de Lyell** (CMH I) +++QCM
    Les LT CD8 **sécrètent des perforines** → perforent la membrane des kératinocytes → nécrose +++
    >

## 5️⃣ Autres Cellules de l'Immunité Cutanée

| Cellule | Rôle | Particularité |
| --- | --- | --- |
| **Lymphocytes NK** | Élimination des cellules du **soi avec défaut de CMH** | Défaut CMH = signal d'infection virale ou cancer → NK s'active |
| **PNN** | Rôle dans l'**inflammation aiguë** anti-infectieuse | Premiers arrivés sur le site d'infection |
| **Macrophages** | **Phagocytose** des débris cellulaires + présentation Ag aux LT | Cellules "poubelles" + CPA secondaires |

> 💡 **Lymphocytes NK** = "tueurs naturels" → reconnaissent l'absence de CMH (que la cellule normale exprime toujours) → si une cellule n'exprime pas le CMH = anormale (infectée ou tumorale) → NK la détruit +++QCM

## 6️⃣ Rôle Immunitaire des Kératinocytes +++

> ⚠️ Les kératinocytes **ne sont pas que des cellules de structure** → ils constituent le **premier rempart de l'immunité innée** +++QCM
> 

| Rôle | Détail |
| --- | --- |
| **Protection mécanique** | Barrière physique contre les agents extérieurs (microbes, allergènes, UV) |
| **Production de PAM** | Peptides Antimicrobiens → notamment les **défensines** = antibiotiques naturels |
- Les PAM / Défensines
    
    **PAM = Peptides Antimicrobiens**
    
    - Produits **directement par les kératinocytes**
    - Les **défensines** sont les PAM les plus importants de la peau
    - Agissent comme des **antibiotiques naturels** : détruisent les bactéries, champignons, certains virus
    - Font partie de l'immunité **innée** (pas de spécificité, pas de mémoire)
    - Leur production peut être **augmentée en cas d'infection** → réponse rapide
    
    > 💡 **Défensines** = PAM produits par les kératinocytes → antibiotiques naturels de la peau +++QCM
    💡 Les kératinocytes sont des acteurs **ACTIFS** de l'immunité innée, pas de simples spectateurs +++
    >

## 7️⃣ Eczéma Allergique de Contact — Les 3 Phases +++

> **Médié par un haptène** = molécule de petite taille, normalement non irritante, mais capable d'activer l'immunité innée (perçue comme un "danger")
**3 phases : Sensibilisation → Déclenchement → Régulation**
> 
- Phase 1 — Sensibilisation (1ère exposition, silencieuse)
    
    ```
    ÉTAPE 1 : Pénétration de l'Ag (haptène) dans l'épiderme
                   │
                   ▼
    ÉTAPE 2 : Reconnaissance par la Cellule Dendritique (CD)
              + Migration dermique
                   │
                   ▼
    ÉTAPE 3 : Migration vers le vaisseau lymphatique afférent
                   │
                   ▼
    ÉTAPE 4 : Arrivée au ganglion lymphatique
                   │
                   ▼
    ÉTAPE 5 : Activation LT CD4 ET LT CD8 spécifiques
              + Migration vers le site de stimulation (peau)
    ```
    
    > 💡 Lors de la sensibilisation, **pas d'eczéma visible** → la personne est sensibilisée mais ne le sait pas encore → réponse silencieuse +++QCM
    > 
- Phase 2 — Déclenchement (2ème exposition et suivantes)
    
    ```
    ÉTAPE 6 : Pénétration de l'Ag + Reconnaissance par CD
                   │
                   ▼
    ÉTAPE 7 : Présentation aux LT spécifiques épithéliaux via CMH
                   │
                   ▼
    ÉTAPE 8 : Réponse adaptative RAPIDE et efficace
              → Production de cytokines (LT helper)
                OU cytotoxicité directe (LT CD8 cytotoxique)
              → Recrutement de PNN et LT spécifiques
              → Apoptose des kératinocytes
                   │
                   ▼
            « TEMPÊTE CYTOKINIQUE »
                   │
                   ▼
                ECZÉMA
    ```
    
    > ⚠️ **"Tempête cytokinique"** = libération massive de cytokines (IL-10, TGFβ notamment) → apoptose des kératinocytes → eczéma visible +++QCM
    > 
- Phase 3 — Régulation
    
    ```
    ÉTAPE 9 : Recrutement de LT régulateurs (Treg)
                   │
                   ▼
    Production d'IL-10 et TGFβ
                   │
                   ▼
    Freinage progressif de la réponse inflammatoire
                   │
                   ▼
    Résolution de l'eczéma
    ```
    
    > 💡 **IL-10 et TGFβ** = cytokines **anti-inflammatoires** des Treg → essentielles pour la résolution de l'eczéma +++QCM
    > 
- Tableau récapitulatif des 9 étapes
    
    
    | N° | Phase | Événement |
    | --- | --- | --- |
    | **1** | Sensibilisation | Pénétration de l'Ag |
    | **2** | Sensibilisation | Reconnaissance par CD + migration dermique |
    | **3** | Sensibilisation | Vaisseau lymphatique afférent |
    | **4** | Sensibilisation | Ganglion lymphatique |
    | **5** | Sensibilisation | Activation LT CD4 + CD8 → migration vers la peau |
    | **6** | Déclenchement | Re-pénétration de l'Ag + reconnaissance CD |
    | **7** | Déclenchement | Présentation aux LT épithéliaux via CMH |
    | **8** | Déclenchement | Cytokines + cytotoxicité → apoptose kératinocytes → **eczéma** |
    | **9** | Régulation | Recrutement Treg → IL-10 + TGFβ → freinage |

## 8️⃣ CMH I vs CMH II — Récapitulatif +++

|  | **CMH I** | **CMH II** |
| --- | --- | --- |
| **Exprimé par** | **Toutes** les cellules nucléées | CPA **professionnelles** uniquement (CD, macrophages, LB) |
| **Présente** | Antigènes **endogènes** (intracellulaires, peptides du soi) | Antigènes **exogènes** (capturés de l'extérieur) |
| **Reconnu par** | **LT CD8** (cytotoxiques) | **LT CD4** (helper / régulateurs) |
| **Pathologies cutanées** | Allergie médicamenteuse, toxidermies, **Lyell** | Psoriasis, eczéma, dermatite atopique |

> 💡 **Règle d'or +++QCM** : CD**4** = CMH I**I** (même chiffre 4→II) / CD**8** = CMH **I** (CD8 "8" → CMH I "1")
>

## 🔟 Mnémotechniques +++

### Immunité innée vs adaptative

```
INNÉE   = Rapide, Peu spécifique, Pas de mémoire  → « RPP »
ADAPTATIVE = Lente, Spécifique, Mémoire          → « LSM »
```

### Les sous-types Th et leurs cytokines

```
Th1  → 1NFγ + IL-2          → 1 chiffre → réponse cellulaire
Th2  → IL-4, 5, 13          → atopie, allergie, eczéma
Th17 → IL-17, 22            → 17, psoriasis, fongique
Treg → IL-10 + TGFβ         → FREIN anti-inflammatoire
```

### CMH → lymphocyte cible

```
CMH I  → CD8  (cytotoxique → Perforine + Granzyme)
CMH II → CD4  (helper Th1/Th2/Th17 ou Treg)

Moyen : CD4 = CMH II (même chiffre pair)
        CD8 = CMH I  (CD8 → détruit, CMH I → toutes les cellules)
```

### Treg — identifier

```
CD4+ CD25+
Produit IL-10 et TGFβ (PAS d'IL-2)
Inhibe LTh via CTLA-4
```

### Les 3 phases de l'eczéma

```
S - D - R
Sensibilisation (silencieuse, 1ère exposition)
Déclenchement  (eczéma visible, tempête cytokinique)
Régulation     (Treg → IL-10 + TGFβ → résolution)
```

### Perforine vs Granzyme (LT CD8)

```
PERFORINE → PERFORE la membrane (porte d'entrée)
GRANZYME  → détruit l'ADN (mort par apoptose)
```
