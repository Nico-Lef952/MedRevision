# I. Principe

➡️ **Test diagnostique d’allergie immédiate** (hypersensibilité de type I).

- Recherche une réaction vis-à-vis d’**allergènes** :
    - **Pneumo-allergènes** (acariens, pollens, animaux…).
    - **Trophallergènes** (aliments).
- Les **mastocytes déjà sensibilisés** se **dégranulent** → réaction cutanée locale rapide.

👉 Explore donc la **réactivité allergique IgE-médiée**.

# II. Technique

- Réalisé sur la **face antérieure de l’avant-bras**.
- **Micro-piqûre cutanée** à travers une goutte de solution allergénique.
- Atteinte **limitée à l’épiderme**.
- **Lecture à 15–20 minutes**.

### Témoins obligatoires

- **Témoin positif** : histamine → doit réagir.
- **Témoin négatif** : eau ou sérum salé → ne doit pas réagir.

👉 Garantit la **validité du test**.

# III. Interprétation

- Mesure de :
    - **Papule** (diamètre).
    - **Réaction inflammatoire locale** (érythème).

➡️ Réaction positive = **sensibilisation allergique** à l’allergène testé.

# IV. Indications en pneumologie

- **Bilan allergologique d’un asthme**.
- Recherche d’un **terrain atopique** dans les pathologies respiratoires.

👉 Permet d’orienter :

- l’**éviction allergénique**,
- le **traitement de fond**,
- éventuellement une **désensibilisation**.
