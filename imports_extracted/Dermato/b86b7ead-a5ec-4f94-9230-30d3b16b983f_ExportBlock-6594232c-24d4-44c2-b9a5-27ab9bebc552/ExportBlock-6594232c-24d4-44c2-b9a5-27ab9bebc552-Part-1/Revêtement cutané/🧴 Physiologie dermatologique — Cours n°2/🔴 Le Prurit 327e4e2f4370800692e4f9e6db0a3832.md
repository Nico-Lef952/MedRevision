## 📌 À retenir absolument

> 💡 **Prurit** = "sensation déplaisante qui provoque le désir de gratter" — peau + semi-muqueuses (**OGE+++**)
💡 **Prurit ≠ Douleur** → différences clés : chaleur exacerbe le prurit (calme la douleur) · morphiniques exacerbent le prurit +++QCM
💡 **Médiateur principal** = **histamine** — mais **n'intervient pas dans tous les types** de prurit +++QCM
💡 **3 types de pruricepteurs** : Rc à l'histamine · Rc **PAR-2** (activé par la tryptase) · Rc du **GRP**
💡 **Voie de transmission** : pruricepteur → moelle épinière → thalamus → zones cérébrales sensorielles, émotionnelles et motrices
💡 **4 types de prurit** selon l'organe responsable : Pruriceptif · Neurogène · Neuropathique · Psychogène
💡 **Prurit pruriceptif** = forme classique d'origine cutanée (ex : urticaire, gale, eczéma)
💡 **Antihistaminiques** = efficaces uniquement dans le prurit à histamine (urticaire) → **inefficaces** dans les autres types +++QCM
💡 **Prurit chronique** → lésions de grattage + **prurigo** + **lichénification**
💡 Causes générales : dermatoses inflammatoires · accumulation de toxines (IRC) · hémopathies (Hodgkin) · endocrinopathies
>

## 1️⃣ Définition & Localisation

> **Prurit** = *"sensation déplaisante qui provoque le désir de gratter"*
> 

| Caractéristique | Détail |
| --- | --- |
| **Localisation** | Peau + **semi-muqueuses** (**OGE+++**) |
| **Causes principales** | Dermatoses inflammatoires · accumulation de toxines · hémopathies · endocrinopathies |
| **Distinction fondamentale** | **Prurit ≠ Douleur** (deux sensations distinctes avec voies et comportements différents) |

---

## 2️⃣ Prurit vs Douleur — Tableau Comparatif +++

| Critère | **Prurit** | **Douleur** |
| --- | --- | --- |
| **Geste en conséquence** | **Grattage** | Retrait |
| **Localisation** | Peau, semi-muqueuses | Ubiquitaire |
| **Effets de la chaleur** | **Exacerbation** | Calme |
| **Effets du froid** | **Calme** | Exacerbation |
| **Effets des morphiniques** | **Exacerbation** +++QCM | Calme |
| **Effets des anti-histaminiques** | Souvent favorables | Aucun |
| **Seuil minimal** | Stimuli **minimes** | Stimuli moyens |

> ⚠️ **Piège majeur +++QCM** : la **chaleur exacerbe le prurit** (et calme la douleur) · les **morphiniques exacerbent le prurit** (et calment la douleur) → inversé par rapport à la douleur !
>

## 3️⃣ Physiopathologie du Prurit

- Médiateurs du prurit
    
    **Plusieurs médiateurs** peuvent être impliqués selon le type de prurit :
    
    | Médiateur | Particularité |
    | --- | --- |
    | **Histamine** | Le + connu · **N'intervient PAS dans tous les types** de prurit +++QCM |
    | **Tryptase** | Active le Rc PAR-2 sur les pruricepteurs |
    | **GRP** (Gastrin-Releasing Peptide) | Médiateur spécifique de certains pruricepteurs |
    | **IL-4, IL-31** | Cytokines prurigènes (ex : dermatite atopique) — au niveau des neurones +++ |
    
    > 💡 Ce sont **IL-4 et IL-31** qui donnent la sensation de démangeaisons au niveau des neurones dans la dermatite atopique → cibles thérapeutiques émergentes +++
    > 
- Les 3 types de pruricepteurs +++
    
    
    | Pruricepteur | Activé par |
    | --- | --- |
    | **Rc à l'histamine** | Histamine (libérée par mastocytes) |
    | **Rc PAR-2** | Tryptase (sérine protéase) |
    | **Rc du GRP** | Gastrin-Releasing Peptide |
    
    > 💡 **Pruricepteur** = terminaison nerveuse spécialisée dans la détection du prurit (≠ nocicepteurs de la douleur)
    > 
- Voie de transmission du prurit
    
    ```
    Stimulus pruritogène (histamine, tryptase, GRP...)
                  │
                  ▼
    Activation du PRURICEPTEUR
    (terminaison nerveuse cutanée)
                  │
                  ▼
          Moelle Épinière (ME)
                  │
                  ▼
               THALAMUS
                  │
                  ▼
    Zones cérébrales :
    ┌─────────────────┐ ┌──────────────────┐ ┌───────────────┐
    │   SENSORIELLES  │ │  ÉMOTIONNELLES   │ │   MOTRICES    │
    │  (où ça gratte) │ │ (désagrément)    │ │ (gratter)     │
    └─────────────────┘ └──────────────────┘ └───────────────┘
    ```

## 4️⃣ Classification du Prurit selon l'Organe Responsable +++

| Type | Organe responsable | Mécanisme | Exemples |
| --- | --- | --- | --- |
| **Pruriceptif** | **Peau** (origine cutanée) | Activation des pruricepteurs cutanés | Gale, urticaire, eczéma, psoriasis ← **forme classique** |
| **Neurogène** | **SNC** | Substance agissant sur le système nerveux central | Prurit aux opioïdes, cholestase (sels biliaires sur SNC) |
| **Neuropathique** | **SNP** | Maladie du système nerveux périphérique | Zona, neuropathie, névralgie post-zostérienne |
| **Psychogène** | **Psychisme** | Manifestation somatique d'un état psy | Dépression, anxiété, troubles somatoformes |

> 💡 **Mémo : P-N-N-P** → Pruriceptif · Neurogène · Neuropathique · Psychogène
⚠️ Les **antihistaminiques ne sont efficaces que dans le prurit pruriceptif à histamine** (urticaire) → **inefficaces** dans les autres types +++QCM
💡 Si brûlure spontanée génitale, zona, grattage → **antidépresseurs** qui agissent sur la transmission neuronale
>

## 5️⃣ Interrogatoire Discriminatif +++

> **L'interrogatoire est fondamental** pour orienter le diagnostic étiologique du prurit
> 

| Élément à rechercher | Précisions |
| --- | --- |
| **Date et mode de début** | Brutal ou progressif |
| **Facteurs déclenchants** | Stress, irritants, contact avec substance... |
| **Évolution** | Aiguë, paroxystique ou chronique |
| **Chronologie** | Heure de la journée, période de l'année (saisonnalité) |
| **Intensité** | Gêne dans le travail, la vie quotidienne, la vie affective, le **sommeil** |
| **Topographie et extension** | Localisé ou généralisé |
| **Facteurs aggravants** | Hypersudation, sport, bains, douches, repas |
| **Facteurs calmants** | Froid, détente |
| **Contexte associé** | Maladies connues, prise de toxiques |
| **Lien avec signes objectifs** | Prurit avant, pendant ou après les signes cutanés |
| **Prurit collectif** | Existence ou non → oriente vers une cause infectieuse/parasitaire (gale+++) |
| **Effets des traitements** | Réponse aux antihistaminiques, corticoïdes... |

> 💡 **Prurit collectif** (plusieurs personnes dans l'entourage) → penser à la **gale** +++ (Sarcoptes scabiei)
>

## 6️⃣ Examen Physique

| Lésion retrouvée | Signification |
| --- | --- |
| **Lésions de grattage** | Prurit aigu ou chronique |
| **Prurigo** | Prurit **chronique** +++ (papules excoriées) |
| **Lichénification** | Prurit **chronique** +++ (épaississement cutané par grattage répété) |
| **Lésions élémentaires** d'une dermatose | Orienter vers une cause cutanée primitive |

> ⚠️ **Prurigo et lichénification = marqueurs de chronicité** du prurit +++QCM
💡 À l'examen physique, on recherche **soit une lésion de grattage, soit une maladie de peau** sous-jacente
>

## 7️⃣ Causes du Prurit — Vue d'ensemble

- Causes cutanées (prurit pruriceptif)
    
    
    | Cause | Exemples |
    | --- | --- |
    | **Dermatoses inflammatoires** | Eczéma, psoriasis, dermatite atopique, urticaire |
    | **Infections / parasitoses** | **Gale** (prurit collectif++), dermatophytoses |
    | **Sécheresse cutanée (xérose)** | Fréquent chez le sujet âgé |
- Causes générales (prurit sine materia = sans lésion cutanée)
    
    
    | Cause | Mécanisme / Détail |
    | --- | --- |
    | **Accumulation de toxines** | Insuffisance rénale chronique (IRC) → urémie → prurit urémique |
    | **Hémopathies** | **Maladie de Hodgkin** +++ (prurit = signe révélateur) · polyglobulie |
    | **Endocrinopathies** | Hyperthyroïdie, hypothyroïdie, diabète |
    | **Cholestase** | Accumulation de sels biliaires → prurit cholestatique |
    | **Médicaments** | Morphiniques, AINS, certains antibiotiques |
    | **Psychogène** | Dépression, anxiété |
    
    > ⚠️ **Prurit sine materia** (sans lésion cutanée) = toujours chercher une cause générale systémique +++QCM
    💡 **Maladie de Hodgkin** = hémopathie classiquement associée au prurit intense +++QCM
    >

## 8️⃣ Mnémotechniques +++

### Les 4 types de prurit

```
P - N - N - P
Pruriceptif  (cutané, forme classique)
Neurogène    (SNC, opioïdes)
Neuropathique (SNP, zona)
Psychogène   (psy, dépression, anxiété)
```

### Prurit vs Douleur — inversions à retenir

```
CHALEUR   → EXACERBE le prurit / CALME la douleur  ← PIÈGE +++
FROID     → CALME le prurit / EXACERBE la douleur
MORPHINE  → EXACERBE le prurit / CALME la douleur  ← PIÈGE +++
ANTIHISTAMINIQUE → efficace sur prurit histaminique / aucun effet sur douleur
```

### Les 3 pruricepteurs

```
H  → Rc à l'Histamine
P  → Rc PAR-2 (tryptase)
G  → Rc du GRP

Moyen : « H-P-G » = « Histamine Provoque le Grattage »
```

### Lésions de chronicité

```
PRURIGO + LICHÉNIFICATION = prurit CHRONIQUE +++
(pas présents dans le prurit aigu)
```

### Interrogatoire discriminatif — éléments clés

```
D - F - E - C - I - T - F - C - L - P - E
Début · Facteurs déclenchants · Évolution · Chronologie
Intensité · Topographie · Facteurs aggravants/calmants
Contexte · Lien avec signes cutanés · Prurit collectif · Effets traitements
```

- ✅ **Morphiniques** → **exacerbent** le prurit (calment la douleur) +++QCM

- ✅ **3 pruricepteurs** : Rc histamine · Rc PAR-2 (tryptase) · Rc GRP +++QCM
- ✅ **Voie** : pruricepteur → ME → thalamus → zones sensorielles/émotionnelles/motrices +++
- ✅ **4 types** : Pruriceptif · Neurogène · Neuropathique · Psychogène +++QCM
- ✅ **Antihistaminiques** = efficaces **uniquement** dans le prurit histaminique (urticaire) → **inefficaces** ailleurs +++QCM
- ✅ **IL-4 et IL-31** = cytokines prurigènes au niveau des neurones (dermatite atopique) +++QCM
- ✅ **Prurigo + lichénification** = marqueurs de prurit **chronique** +++QCM
- ✅ **Prurit collectif** → penser **gale** +++ (Sarcoptes scabiei)

- ✅ **PIÈGE 2** — Prurit sine materia (sans lésion cutanée) → toujours chercher **cause systémique** (hémopathie, endocrinopathie, IRC, cholestase) +++
