## 📌 À retenir absolument

> 💡 **2 questions préalables** : 1) L'indication est-elle pertinente ? 2) Où biopsier ?
💡 **Pathologie inflammatoire NON bulleuse** → lésion **récente** et non remaniée +++QCM
💡 **Pathologie bulleuse** → biopsie **à cheval** sur peau saine + peau décollée +++QCM
💡 **Anesthésie** = Lidocaïne 0,5 / 1 / 2% +/- adrénaline · injection **hypodermique**
💡 **Punch** = lame cylindrique, lésions petites · **Bistouri** = prélèvements larges ou hypoderme
💡 **Histologie standard** → **Formol 10%** ← JAMAIS l'IFD dans le formol +++QCM
💡 **IFD** → sérum physiologique / azote liquide OU **Liquide de Michel** ← tissu "vivant" obligatoire +++QCM
💡 **IFD** = anticorps fixés **dans le tissu** (biopsie) · **IFI** = anticorps **circulants** (sérum/sang) +++QCM
💡 **IFD Pemphigus** = fluorescence en **maille de filet** (intercellulaire) +++QCM
💡 **IFD Pemphigoïde bulleuse** = fluorescence **linéaire** le long de la membrane basale +++QCM
💡 **ELISA** = identification précise de l'antigène cible + quantification → suivi thérapeutique +++
💡 **Renseignements cliniques** = FONDAMENTAUX à fournir à l'anatomopathologiste +++
💡 **Modèle psoriasiforme** = allongement papilles dermiques → psoriasis · **Spongiotique** = œdème papilles → eczéma · **Lichénoïde** = nécrose kératinocytes basaux → lichen, lupus, GVH

## 1️⃣ Préalables — 2 Questions Obligatoires

> Avant toute biopsie cutanée, **2 questions cliniques sont impératives** :
> 

| Question | Détail |
| --- | --- |
| **1. L'indication est-elle pertinente ?** | Est-ce que l'examen apportera la réponse diagnostique ? Certaines dermatoses se diagnostiquent cliniquement (ex : eczéma) → biopsie inutile |
| **2. Où biopsier ? (CRUCIAL)** | Quelle lésion va m'apporter l'information maximale ? |
- Choix du site de biopsie selon la pathologie +++
    
    
    | Type de pathologie | Site à biopsier | Raison |
    | --- | --- | --- |
    | **Inflammatoire NON bulleuse** | Lésion **récente** et **non remaniée** (pas de croûtes, pas grattée) | Lésion ancienne ou modifiée = histologie non interprétable |
    | **Bulleuse** | **À cheval** sur peau saine ET peau décollée (zone de clivage) | Capturer la zone de décollement pour analyser le plan de clivage |
    | **Tumorale** | Zone la plus **représentative** de la lésion | Éviter les zones nécrotiques ou remaniées en périphérie |
    
    > ⚠️ **Ne jamais biopsier une lésion vieille, en train de cicatriser, ou modifiée par le traitement** +++QCM
    >

## 2️⃣ Déroulement de la Biopsie

- Procédure complète
    
    
    | Étape | Détail |
    | --- | --- |
    | **1. Expliquer au patient** | Consentement éclairé +++ |
    | **2. Lavage des mains** | Hygiène de base |
    | **3. Port de gants** | **Non stériles** (suffisant pour biopsie cutanée) |
    | **4. Désinfection locale** | Nettoyage du site de prélèvement |
    | **5. Anesthésie locale** | Injection **hypodermique** de **Lidocaïne 0,5 / 1 / 2%** +/- adrénaline |
    | **6. Prélèvement** | Au punch ou au bistouri |
    | **7. Suture** | **Systématique** — ne pas oublier +++QCM |
- Choix de l'outil
    
    
    | Outil | Technique | Indications |
    | --- | --- | --- |
    | **Punch** | Lame cylindrique rotative | Lésions de **petite taille**, standard, rapide |
    | **Bistouri** (lame froide) | Incision elliptique | Prélèvements **plus larges** ou atteinte de l'**hypoderme** |
    
    ```
    BISTOURI = nécessaire pour atteindre l'HYPODERME +++QCM
    PUNCH = standard pour lésions de petite taille
    TOUJOURS SUTURER après le prélèvement +++
    ```

## 3️⃣ Conservation et Transport +++

> **Le mode de conservation dépend de l'analyse demandée** — erreur de conservation = prélèvement inutilisable
> 

| Analyse | Conservation | Règle absolue |
| --- | --- | --- |
| **Histologie standard** | **Formol 10%** | — |
| **IFD (Immunofluorescence Directe)** | Compresse stérile imbibée de **sérum physiologique** → congélation **azote liquide** OU **Liquide de Michel** | **JAMAIS dans le Formol** +++QCM |
| **Bactériologie/Mycologie** | Milieu spécifique selon germe recherché | — |

> ⚠️ **PIÈGE MAJEUR +++QCM** : L'IFD nécessite un tissu **"vivant"** (non fixé) → **INTERDICTION ABSOLUE de mettre dans le Formol** → détruit les anticorps et rend l'examen ininterprétable
> 

```
MÉMO CONSERVATION :
Histologie  → FORMOL 10%
IFD         → Sérum phy + azote / Liquide de Michel (JAMAIS formol)
```

## 4️⃣ Renseignements Cliniques +++

> ⚠️ **Les renseignements cliniques sont FONDAMENTAUX** et doivent accompagner SYSTÉMATIQUEMENT la biopsie +++QCM
> 

Ils permettent à l'anatomopathologiste d'**orienter son analyse** → sans eux, interprétation impossible sur les lésions inflammatoires.

**À noter sur la demande :**

- Hypothèse(s) diagnostique(s)
- Topographie et nature du prélèvement
- Traitement en cours (peut modifier l'aspect histologique)
- Ancienneté de la lésion

## 5️⃣ Les Lésions Histologiques par Niveau +++

- Lésions épidermiques
    
    
    | Terme | Définition |
    | --- | --- |
    | **Acanthose** | Épaississement de tout ou partie (crêtes) de l'épiderme |
    | **Acantholyse** | Perte de cohésion des kératinocytes par rupture des **desmosomes** → fentes ou bulles **intra-épidermiques** |
    | **Atrophie** | Amincissement global de l'épiderme par diminution du nombre de couches cellulaires |
    | **Épidermotropisme** | Migration de **cellules tumorales** dans l'épiderme |
    | **Exocytose** | Migration de **cellules inflammatoires** dans l'épiderme |
    | **Hypergranulose** | Épaississement de la couche granuleuse |
    | **Hyperkératose** | Épaississement de la couche cornée |
    | **Nécrose kératinocytaire** | Mort cellulaire avec aspect d'apoptose ou de vacuolisation |
    | **Papillomatose** | Accentuation du dessin des papilles et des crêtes inter-papillaires |
    | **Parakératose** | Persistance de structures nucléaires dans les cornéocytes |
    | **Pustule** | Conséquence d'une spongiose avec PNN ou éosinophiles (uni ou multiloculaire) |
    | **Spongiose** | **Œdème intercellulaire** de l'épiderme |
    | **Vésicule** | Cavité formée par la confluence de zones spongiotiques |
    
    > 💡 **Épidermotropisme** (tumorale) ≠ **Exocytose** (inflammatoire) → piège de vocabulaire +++QCM
    💡 **Acantholyse** = rupture des desmosomes → bulles intra-épidermiques (ex : Pemphigus) +++QCM
    > 
- Lésions dermo-épidermiques — Les 4 Grands Modèles +++
    
    
    | Modèle | Description histologique | Exemples cliniques |
    | --- | --- | --- |
    | **Psoriasiforme** | Allongement **parallèle et régulier** des papilles dermiques, toit aminci · Inflammation dans les papilles + exocytose | **Psoriasis** |
    | **Spongiotique** | **Œdème des papilles** + inflammation du derme superficiel + exocytose | **Eczéma**, dermite séborrhéique |
    | **Lichénoïde** (dermite d'interface) | **Vacuolisation et/ou nécrose** des kératinocytes basaux + infiltrat sous-épidermique + exocytose basale = aspect basale **"grignotée"** | **Lichen plan**, toxidermies lichénoïdes, **GVH**, **lupus** |
    | **Bulle sous-épidermique** | **Décollement** de la jonction dermo-épidermique + infiltrat inflammatoire (nature oriente le diagnostic) | **Pemphigoïde bulleuse**, toxidermie bulleuse |
    
    ```
    MÉMO 4 MODÈLES DERMO-ÉPIDERMIQUES +++
    
    PSORIASIFORME → papilles allongées régulières → PSORIASIS
    SPONGIOTIQUE  → œdème papilles + exocytose   → ECZÉMA
    LICHÉNOÏDE    → nécrose kératinocytes basaux  → LICHEN / GVH / LUPUS
    BULLEUX       → décollement jonction DE        → PEMPHIGOÏDE BULLEUSE
    ```
    
- Lésions dermiques
    
    
    | Terme | Définition |
    | --- | --- |
    | **Atrophie dermique** | Diminution d'épaisseur du derme + raréfaction des fibres de collagène |
    | **Élastolyse** | Disparition des fibres élastiques |
    | **Élastose (sénile)** | Modifications du collagène liées à l'action des UV |
    | **Incontinence pigmentaire** | Présence de mélanine dans le derme superficiel (libre ou dans des mélanophages) |
    | **Infiltrats cellulaires** | Présence de cellules inflammatoires ou tumorales dans le derme |
    | **Œdème dermique** | Pâleur du derme secondaire à la présence de liquide interstitiel |
    | **Nécrobiose** | Altération des fibres de collagène → éosinophiles et amorphes |
    | **Sclérose dermique** | Épaississement des fibres de collagène → aspect hyalin |
- Lésions hypodermiques
    
    
    | Terme | Définition |
    | --- | --- |
    | **Adiponécrose** | Nécrose des adipocytes → images de membranes et kystes |
    | **Atrophie hypodermique** | Diminution du tissu adipeux |
    | **Lipophagie** | Résorption graisseuse par des histiocytes-macrophages → aspect spumeux |
    | **Panniculite lobulaire** (panniculite vraie) | Inflammation **primitive des lobules** adipeux |
    | **Hypodermite septale** | Inflammation localisée aux **septas** |
    | **Hypodermite mixte** | Inflammation lobulaire **ET** septale |
    
    ```
    INFLAMMATION HYPODERMIQUE :
    Panniculite  = lobules (primitive)
    Septale      = septas seulement
    Mixte        = lobules + septas
    ```

## 6️⃣ FOCUS — Immunofluorescence (IF) +++

- IFD — Immunofluorescence Directe (sur la biopsie)
    
    > **"Montre que la maladie est bien auto-immune et localise les dépôts dans le tissu"**
    > 
    
    | Caractéristique | Détail |
    | --- | --- |
    | **Support** | **Biopsie cutanée** (tissu vivant, non fixé) |
    | **Principe** | Application d'anticorps fluorescents sur la coupe → se fixent sur les auto-Ac déjà présents dans le tissu du patient |
    | **Ce qu'elle détecte** | Dépôts d'**IgG, IgA, C3** fixés **dans le tissu** |
    | **Conservation** | Sérum phy + azote liquide **OU Liquide de Michel** → **JAMAIS formol** |
    | **But** | **Diagnostic de certitude** — preuve histologique de l'auto-immunité |
    | **Limite** | Ne dit **pas quel est l'antigène cible** précis |
    
    **Résultats typiques +++QCM :**
    
    | Pathologie | Aspect IFD |
    | --- | --- |
    | **Pemphigus vulgaire** | Fluorescence en **"maille de filet"** / **résille** (intercellulaire, intra-épidermique) |
    | **Pemphigoïde bulleuse** | Fluorescence **linéaire** le long de la **membrane basale** (jonction dermo-épidermique) |
    
    > 💡 **Résille/filet** = Pemphigus (intra-épidermique) · **Linéaire** = Pemphigoïde (membrane basale) +++QCM
    > 
- IFI — Immunofluorescence Indirecte (sur le sang)
    
    > **"Détecte les anticorps circulants dans le sang — utile pour le diagnostic, l'orientation et surtout le SUIVI"**
    > 
    
    | Caractéristique | Détail |
    | --- | --- |
    | **Support** | **Sérum du patient** (sang, tube sec) |
    | **Principe** | Sérum du patient mis en contact avec une peau-test (œsophage de singe, peau de rat) → révélation des Ac circulants par marqueur fluorescent |
    | **Ce qu'elle détecte** | **Anticorps circulants** dans le sang |
    | **But** | Confirmation diagnostique + **suivi de l'activité de la maladie** |
    
    **Pourquoi faire une IFI si on a déjà une IFD ?**
    
    | Raison | Explication |
    | --- | --- |
    | **Suivi de l'activité** | Le titre des Ac circulants corrèle avec l'activité clinique · Baisse = réponse au traitement · Remontée = risque de rechute |
    | **Orientation diagnostique plus fine** | Distingue les anti-SIC (pemphigus) des anti-membrane basale (pemphigoïde) |
    | **Dépistage pré-clinique** | Peut être positive **avant** l'apparition des lésions cutanées |
    | **Diagnostic différentiel** | IFI négative + IFD positive → forme **limitée** ou **traitée** |
    
    > 💡 **L'IFI n'est PAS redondante avec l'IFD** → elle apporte une dimension **dynamique** (suivi) et **systémique** (Ac circulants) que l'IFD ne donne pas +++QCM
    > 
- ELISA — Enzyme-Linked Immunosorbent Assay
    
    > **"Identification précise de l'antigène cible avec quantification"**
    > 
    
    | Caractéristique | Détail |
    | --- | --- |
    | **Support** | Sérum du patient |
    | **Principe** | Détection colorimétrique des Ac spécifiques contre un antigène précis |
    | **But** | Identifier l'**antigène spécifique** + **quantification** → suivi thérapeutique |
    | **Avantages** | Très spécifique · Quantitatif · Standardisé · Automatisé · Reproductible |
    
    **Antigènes cibles selon la maladie +++QCM :**
    
    | Antigène | Maladie |
    | --- | --- |
    | **Anti-desmogléine 3 (Dsg3)** | **Pemphigus vulgaire** |
    | **Anti-BP180** | **Pemphigoïde bulleuse** |
    | **Anti-ADN natif** | **Lupus** |
- Tableau comparatif IFD / IFI / ELISA +++
    
    
    | Caractéristique | **IFD** | **IFI** | **ELISA** |
    | --- | --- | --- | --- |
    | **Support** | **Biopsie** (tissu) | **Sérum** (sang) | **Sérum** (sang) |
    | **Cible** | Ac fixés dans le tissu | Ac circulants | Ac spécifiques quantifiés |
    | **Conservation** | Sérum phy / Michel (**pas formol**) | Tube sec standard | Tube sec standard |
    | **But** | Diagnostic de certitude | Confirmation + suivi | Identification Ag + suivi |
    | **Spécificité** | Localise les dépôts | Oriente le groupe | **Très spécifique** (antigène précis) |
    | **Quantitatif ?** | Non | Semi-quantitatif | **Oui** (idéal pour suivi) |

## 7️⃣ Récapitulatif — Quel Examen pour Quel Contexte ?

```
SUSPICION DE DERMATOSE AUTO-IMMUNE (pemphigus, pemphigoïde...) :

→ IFD  (biopsie fraîche, liquide de Michel)
   = confirme l'auto-immunité tissulaire
   = localise les dépôts (filet = pemphigus / linéaire = pemphigoïde)

→ IFI  (sérum)
   = confirme + oriente le groupe diagnostique
   = utile pour le SUIVI de l'activité (titre des Ac)

→ ELISA (sérum)
   = identifie l'antigène précis (Dsg3, BP180, ADN natif...)
   = quantification exacte → suivi thérapeutique optimal

SUSPICION DE DERMATOSE TUMORALE / INFLAMMATOIRE :
→ Histologie standard → FORMOL 10%
```

## 8️⃣ Mnémotechniques +++

### 2 questions avant toute biopsie

```
P - S
Pertinence (l'examen apportera-t-il la réponse ?)
Site      (quelle lésion biopsier ?)
```

### Conservation selon l'examen

```
FORMOL → Histologie standard
MICHEL/AZOTE → IFD (jamais formol !)

Moyen : « F pour Fixé (formol) / M pour Michel (vivant) »
```

### Résultats IFD selon la maladie

```
FILET / RÉSILLE  → PEMPHIGUS  (intra-épidermique)
LINÉAIRE         → PEMPHIGOÏDE (membrane basale)

Moyen : « Filet = PemPhigus · Ligne = Pemphigoïde »
```

### Les 4 modèles dermo-épidermiques

```
P - S - L - B
Psoriasiforme → papilles allongées régulières → Psoriasis
Spongiotique  → œdème papilles + exocytose   → Eczéma
Lichénoïde    → nécrose kératinocytes basaux  → Lichen/GVH/Lupus
Bulleux       → décollement jonction DE        → Pemphigoïde bulleuse
```

### Épidermotropisme vs Exocytose

```
Épidermotropisme = cellules TUMORALES migrent dans épiderme
Exocytose        = cellules INFLAMMATOIRES migrent dans épiderme
```
